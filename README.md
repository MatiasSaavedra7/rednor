
# REDNOR

Proyecto realizado para un emprendimiento dedicado a la venta y alquiler de equipos multifuncion/fotocopiadoras, seguimiento y control de los mismos, como tambien el registro de clientes y un historial de alquileres, donde se podr√°n ver reflejados el ingreso y egreso de los equipos.




## Run Locally

Clone the project

```bash
  git clone https://github.com/MatiasSaavedra7/rednor.git
```

Go to the project directory

```bash
  cd rednor
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
